1. What is this repository for?
This repository includes recursion and iterative process of the Floyd–Warshall algorithm. The algorithm calculates the shortest paths between every pair of vertices.

The main purpose of this project is to demonstrate the three nested loops that can be rewritten using recursion. The repository also contains unit tests and a performance test that compares the recursive and iterative.

2. How do I get set up?
Clone the repository using Git:
git clone <repository-url>

Then move into the project directory:
cd <repository-name>

Alternatively, download the repository as a ZIP file and extract it.

3.Running the scripts
Run commands from the repository's root directory.

4. Requirements
Python 3.13
